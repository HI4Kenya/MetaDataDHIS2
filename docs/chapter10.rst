10: How can i contribute 
==================================

1) fork the repo (to be a contributor)

.. note::  https://github.com/kanji-karanja/MetaDataDHIS2/network/members

2) git clone the repo

.. note:: https://github.com/kanji-karanja/MetaDataDHIS2.git

3) install npm 

.. note:: npm install

4) start development
