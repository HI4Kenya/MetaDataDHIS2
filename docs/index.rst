Metadata Dictionary
==============================

What is it?
===========
A metadata dictionary is a DHIS2 application that gives more information 
on metadata and what it entails.This includes description of metadata thus 
allowing users to view and edit them in real time so as to assist users using DHIS2 in whatever instance to get more detailed information on the metadata they are interested in.

**Metadata**

It is a set of data that describes and gives information about other data.

For our instance metadata refers to the minute information e.g "name" that gives more details about a specific data element or Indicators. 


**DHIS2**

DHIS is an open source software platform for reporting, analysis and dissemination of data for all health programs, developed by the Health Information Systems Programme.
