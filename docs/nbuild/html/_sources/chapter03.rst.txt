3: Navigating through the metadata
==================================

This page gives a detailed guide on how one can navigate and get more information on datasets.

1. On-clicking the Data-sets button an hashed and alphabetical  list of buttons is brought forth.

2. Clicking these buttons a list of Data-sets starting with the corresponding alphabets on the buttons will be shown.
